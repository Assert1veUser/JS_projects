Request (запрос) и Response (ответ) — это два основных механизма взаимодействия между клиентом и сервером в Node.js.

Request представляет собой HTTP-запрос, который отправляется на сервер с помощью метода `http.get` или `https.get`.
Он содержит параметры запроса, такие как URL, заголовки и тело запроса. Когда запрос отправлен, он обрабатывается
сервером, который может вернуть ответ в виде объекта Response или Error.

Response представляет собой объект, который содержит информацию о том, был ли получен запрос и какой статус имеет его
обработка. Он может содержать различные типы данных, такие как строки, объекты, массивы и т.д. В случае успешного
выполнения запроса, ответ будет содержать код состояния 200 OK, а в случае ошибки — код состояния 404 Not Found.

Таким образом, Request и Response являются основными механизмами взаимодействия между клиентом и сервером в Node.js,
которые позволяют обмениваться данными между ними.

------------------------------------------------------------------------------------------------------------------------

"require" — это встроенный модуль Node.js, который позволяет загружать модули из файлов или директорий. Модуль "require"
является ключевым словом в модуле "module.exports", который используется для определения модуля в глобальном
пространстве имен.
